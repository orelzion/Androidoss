תכנית הקורס

הקורס מחולק ל2, קורס יסודות התכנות והבוטקאמפ לפיתוח לאנדרואיד.

יסודות התכנות

מבוא למדעי המחשב ואלגוריתמים

מה זה מחשב? מה זו תוכנה? איך עובד הזיכרון? ואיך כותבים שפת תוכנה?
נלמד גם אלגוריתמים נפוצים ואיך למדוד את היעילות שלהם

מפגש ראשון עם קוטלין

נכיר את שפת התכנות קוטלין - השפה הרשמית לכתיבת קוד בפיתוח לאנדרואיד.
נלמד להכיר את Kotlin Playground, ונבנה את התוכנה הראשונה שלנו בקוטלין.
נתחיל להכיר גם פונקציות ומשתנים ואת הערכים השונים שניתן לתת להם.

לולאות

לכתוב קוד זה כיף, אז למה שלא נעשה את זה שוב ושוב? :)
נלמד איך להשתמש בלולאות כדי לגרום לקוד לחזור על עצמו שוב ושוב. נכיר את לולאת for ואת לולאת while, ונלמד להבין מה זה block בקוטלין ואיך להשתמש בו נכון.

אז איפה העוגה?

נקבל משימה לבנות תוכנה שמציירת עוגה עם נרות כמספר גילו של ילד.
המשימה תעזור לנו להבין טוב יותר איך משתמשים בלולאות ותכין אותנו לשיעור הבא.

אם אז - משפטי תנאי

אף קוד לא מושלם בלי משפטי תנאי.
נכיר את השימוש ב if/else ונלמד גם להשתמש במשפט when הייחודי לקוטלין.

רשימות, מיפוי ופונקציות למדה

כבר למדנו מה הם משתנים ואיך משתמשים בהם. הפעם נלמד על להכניס משתנים לרשימות כדי שיהיה אפשר לקרוא אותם בצורה יעילה.
נלמד איך עובדות רשימות בקוטלין ומה ההבדל בין list לmap.
נצלול פנימה לפונקציות ונבין מה זה למדה ואיך קוטלין משתמשת בזה כדי לקצר לנו את הקוד.

קלאסים ואובייקטים

אנחנו משתמשים בקלאסים כדי לייצג מידע, לוגיקת תוכנה, וכדי לארגן בצורה מסודרת את הקוד שלנו.
נלמד את ההבדל בין class לdata class ומה זה object. ונלמד מתי להשתמש בכל אחד מהם.
נלמד גם מהם properties של קלאס ואיך משתמשים בהם נכון.

נוקשות ויציבות

מתי משתמשים בvar ומתי דווקא בval. מה היתרון של קלאס סגור?
איך להימנע מnull poinet exception שיקריס לנו את האפליקציה?
נלמד גם על scoped functions, immutability, lazy loading, וdelegation.

בוטקאמפ פיתוח לאנדרואיד

גיט וגיטהאב

ניהול קוד וניהול גרסאות זה כלי סופר חשוב בפיתוח תוכנה.
גיט הוא הסטנדרט לניהול קוד בתעשיה, ואנחנו נלמד איך משתמשים בגיט ולמה.
בנוסף גם נפתח חשבון בgithub לשם נעלה את הקוד של האפליקציות שנפתח כדי שמעסיקים עתידיים יוכלו לראות אותו.

אוקיי אנדרואיד

נתחיל להכיר את אנדרואיד, מתי פותחה מערכת ההפעלה ולמה.
נוריד את Android Studio הכלי בו נפתח את האפליקציות שלנו. נלמד איך פותחים פרוייקט פיתוח חדש ונכתוב את האפליקציה הראשונה שלנו!

Resources

layouts הם הבסיס לכתיבת פקדים על המסך. נלמד כמה מהנפוצים ואיך להשתמש בהם נכון.
נלמד מה השימוש של ids בקוד שלנו. איך שומרים טקסטים, תמונות, צבעים וקבצי שפה.

ניווט ומסכים

נלמד איך מגדירים מסכים שונים באנדרואיד. נלמד על activity ועל fragment. מה ההבדלים ביניהם, איך מנווטים ביניהם ועוד.
חשוב מאוד, נלמד להכיר את הlifecycle של מסכים באנדרואיד ומה צריכים לעשות כדי לתמוך בהם.

Navigation Component

נלמד לנווט באמצעות הכלי החדש Navigation Component.
במה הוא שונה מהנייוט המסורתי באפליקציות אנדרואיד. איך משתמשים בו כדי לעבור בין מסכים, ואיך מעבירים מידע ממסך למסך אחר

MVVM and Architecture components

נלמד לא רק איך כותבים קוד, אלא איך כותבים קוד בצורה נכונה, שאפשר לקרוא בקלות, שאפשר לתחזק בקלות ושאפשר לשנות בקלות.
נלמד על החשיבות של ארכיטקטורת תוכנה, ועל ארכיטקטורת MVVM.
נלמד איך לממש MVVM באנדרואיד באמצעות שימוש בview model וב live data

חיבור לשרת מרוחק

כמעט כל אפליקציה משתמשת בשרת כדי לקרוא ולאחסן נתונים.
נלמד איך להתחבר לאינטרנט מתוך אפליקציית אנדרואיד. מה זה api ומה זה rest. איך להשתמש בספריית retrofit כדי להתחבר לapi בצורה מסודרת ונוחה. נלמד לקרוא לכתוב אובייקטי json באמצעות ספריית kotlinx.serialization. ואיך להוריד ולהציג תמונות שמאוחסנות בשרת.

offline first

להתחבר לאינטרנט זה חשוב מאוד, אבל איך אפשר לשמור מידע שהורדנו מהאינטרנט בצורה מקומית, כדי שפעם הבא לא נצטרך לחכות הרבה זמן עד שהוא ירד שוב?
נלמד קצת על מסדי נתונים, איך להשתמש בsqlite ואיך לעשות את זה באנדרואיד באמצעות ספריית Room.

Coroutines

אתה טובים במולטי טאסקינג? מצליחים לנהל כמה משימות במקביל? כשאתם כותבים לאנדרואיד זה נעשה ממש חיוני.
נלמד איך להשתמש בcoroutines כדי לנהל עבודה מקבילית, כך שנוכל למשל לטעון קובץ גדול בלי לתקוע את האפליקציה.

RecyclerView

איך להציג פריטים ברשימה בצורה יעילה? נשמע קל, אבל למען האמת זה לא כל כך מובן מאליו.
נלמד איך להשתמש בrecyclerView בצורה נכונה, איך לעדכן אותו באמצעות DiffUtil כדי לטעון אליו ערכים חדשים בצורה יעילה ואנימטיבית/
רשום אותי!

למי הקורס מתאים?

הקורס מתאים לכל מי שמוכן להשקיע לפחות 8 שעות בשבוע לתרגול. אין דרישות קדם. לא צריך ניסיון בתכנות, הקורס מלמד תכנות מאפס. הדברים היחידים שנדרשים מכם הם

  • מחשב מהשנים האחרונות
  • חיבור יציב לאינטרנט
  • אנגלית ברמה סבירה
כל השיעורים יועברו אונליין ויוקלטו כדי שניתן יהיה לחזור אליהם בהמשך.

מי אני?

אני אוראל ציון מפתח אנדרואיד בתשע השנים האחרונות. עבדתי בכמה חברות סטארטאפ ואני עובד היום ב viz.ai.

במהלך השנים צברתי המון ידע וניסיון, ופגשתי עשרות מתכנתות מתחילות ומתחילים. את הקורס בניתי על בסיס הידע שצברתי, ועל בסיס המסע שלי מאפס ניסיון למפתח אנדרואיד בכיר.
